# hra_kamen_nuzky_papir
Jednoduchá webová aplikace postavená na frameworku Flask, která umožňuje uživateli hrát klasickou hru "Kámen, nůžky, papír" proti počítači přímo v prohlížeči.


Funkce
Výběr tahu pomocí stylových tlačítek.
Počítač generuje své tahy pomocí modulu random.
Logika hry okamžitě určí vítěze, poraženého nebo remízu.
